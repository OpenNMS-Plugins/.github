# OpenNMS Plugins

Home for plugins that extend [OpenNMS](https://www.opennms.com/), the open-source network monitoring and management platform.

All plugins in this organization are **officially maintained and supported by [The OpenNMS Group, Inc.](https://www.opennms.com/)** and released alongside supported versions of OpenNMS Horizon and OpenNMS Meridian. Per-plugin compatibility and installation details live in each repository's `README`.

## Support

- **Commercial** — covered by an active OpenNMS Meridian subscription. See [opennms.com/support](https://www.opennms.com/support/).
- **Community** — open an issue on the relevant repository, or use the channels in [Mattermost](https://chat.opennms.com/opennms/channels/opennms-discussion) or [Discourse](https://opennms.discourse.group/).
- **Security** — follow [opennms.com/security](https://www.opennms.com/security/); do not file public issues.

## Contributing

Pull requests are welcome. Follow each repository's `CONTRIBUTING` guide, use [Conventional Commits](https://www.conventionalcommits.org/), and sign off your commits (DCO). Plugins are released under **Apache-2.0** unless a repository's `LICENSE` says otherwise — contributions to **AGPL-3.0** plugins additionally require a signed [Contributor License Agreement](https://cla-assistant.io/OpenNMS/opennms).
